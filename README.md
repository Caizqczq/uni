# UniLife —— 有你生活，优你生活
## 项目简介
UniLife 是一款面向学生的在线论坛，致力于提升校园生活体验。其口号为 “有你生活，优你生活”，核心功能包括论坛交流、学习资源共享、课程管理、AI 学习辅助等。项目初期以网站形式开发，后续可能扩展为移动端应用。

## 功能特性

### 1.用户系统
+ 账号注册/登录（支持智慧珞珈认证）
+ 个人资料管理

### 2.论坛功能
+ 主题发布/评论/点赞
+ 话题分类（学习、校园生活、兴趣交流等）
+ 接入校园新闻（自动抓取/人工编辑）

### 3.学习资源共享
+ 共享文档（学生自主填写各课程作业信息）
+ 资料上传/下载权限管理

### 4.课程表 & 个人行程安排
+ 课程表导入（API 接入学校系统或手动录入）
+ 个人日程管理（整合课程表，支持自定义编辑）

### 5.AI 辅助学习
+ 学习计划制定（结合课程安排和个人目标）
+ 任务提醒（结合课程表，提供智能通知）
+ 部分课程智能辅助

### 6.查询功能
+ 课程信息查询
+ 资料搜索
+ 论坛帖子搜索

### 7.积分系统
+ 贡献积分（发帖、上传资源等）
+ 兑换功能（待定，可用于解锁高级功能等）
